# Hadeel Alshibani | AI Portfolio

Personal portfolio website for Hadeel Alshibani, a fresh Artificial Intelligence graduate from Princess Nourah bint Abdulrahman University.

The portfolio highlights applied AI projects across computer vision, NLP, RAG systems, multi-agent systems, recommendation systems, and machine learning. It also includes technical skills, experience, certifications, and contact links.

## Featured Projects

* LeafGuard: Agricultural AI platform for plant disease detection, diagnosis, RAG-based guidance, and product recommendation.
* Aminoon: Graduation project focused on AI biodiversity monitoring, extinction risk prediction, habitat suitability modeling, forecasting, and explainable AI.
* SDAIA Agentic RAG: Agentic RAG platform with APIs, secure chat sessions, retrieval workflows, and evaluation dashboards.
* Intelligent Traffic Control System: Multi-agent smart traffic simulation using JADE and FIPA-ACL communication.

## Built With

HTML, CSS, JavaScript, Python, FastAPI, LangChain, ChromaDB, FAISS, scikit-learn, TensorFlow/Keras, and other AI engineering tools.
